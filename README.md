Disskey
====

Misskey client for desktop

This project is being developed and is an experiment more than anything else now.

## Build and run Disskey

1. `git clone git://github.com/AyaMorisawa/Disskey.git`
2. `cd Disskey`
3. `npm install`
4. `npm run dtsm`
5. `npm run build`
6. `npm start`
